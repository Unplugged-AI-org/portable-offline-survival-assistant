POSA includes monaco.map as a tiny Mapsforge-compatible local test fixture.

Source: https://download.mapsforge.org/maps/v5/europe/monaco.map
Downloaded: 2026-07-01
Size at source index: 342K

The fixture is derived from OpenStreetMap-compatible data and is included only
to exercise local offline map rendering during early POSA development. The app
must show OpenStreetMap attribution anywhere this map is rendered.
